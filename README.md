# js-blockchain

js-blockchain is a simple express application skeleton wrapping a blockchain

## Installation
```
$ npm install
$ yarn 
```

## Usage

### Tests

```
$ npm run test
$ yarn test
```

### Development nodemon server 

```
$ npm run dev
$ yarn dev
```

### Run 

```
$ npm run serve
$ yarn serve
```
